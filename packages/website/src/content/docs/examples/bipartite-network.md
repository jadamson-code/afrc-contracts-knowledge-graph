Bipartite (or two-mode) networks contain two distinct sets of nodes, where edges only connect nodes from different sets. Common examples include author-paper networks, user-product relationships, or any dataset with two entity types linked together.

This example will demonstrate how to render a bipartite graph with sigma.js, positioning the two node sets in separate columns and using distinct colors or shapes to differentiate them.
