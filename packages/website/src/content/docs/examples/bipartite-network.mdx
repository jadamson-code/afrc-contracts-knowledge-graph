Bipartite (or two-mode) networks contain two distinct sets of nodes, where edges only connect nodes from different sets. This example renders the [diseasome](https://gephi.org/datasets/diseasome.gexf.zip) dataset — a disease–gene network — with distinct visual treatments for each node type.

## How it works

The graph is loaded from a GEXF file using [graphology-gexf](https://github.com/graphology/graphology-gexf). Each node has a `type` attribute (`"disease"` or `"gene"`) that drives conditional styles.

### Per-type styling with conditional rules

```typescript
styles: {
  nodes: [
    DEFAULT_STYLES.nodes,
    // Disease nodes: circles with bold labels rendered over them
    {
      when: (attrs) => attrs.type === "disease",
      then: {
        shape: "circle",
        labelFont: "bold Arial, sans-serif",
        labelSize: 16,
        labelPosition: "over",
      },
    },
    // Gene nodes: diamonds with small monospace labels at an angle
    {
      when: (attrs) => attrs.type === "gene",
      then: {
        shape: "diamond",
        labelFont: "'Courier New', monospace",
        labelSize: 10,
        labelPosition: "right",
        labelAngle: Math.PI / 4,
      },
    },
  ],
}
```

### Key concepts

- **Multiple shapes**: declare available shapes in `primitives.nodes.shapes` using factory functions (here `[sdfCircle(), sdfDiamond()]`)
- **Conditional styles**: use `when` predicates on node attributes to style different node types differently
- **Label customization**: per-node label font, size, position, and angle
- **GEXF import**: use `graphology-gexf` to parse GEXF files directly
