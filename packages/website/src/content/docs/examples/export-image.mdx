This example shows how to use the `@sigma/export-image` package to export a rendered graph as a PNG or JPEG image. You can configure the output format and optionally reset the camera state before exporting.

<SigmaExample id="cookbook/export-image" height="500px" />

## How it works

### 1. Build a graph and render it

A small graph of people, cities, and foods is created with a force-directed layout running in a web worker:

```typescript
import { downloadAsImage } from "@sigma/export-image";
import ForceSupervisor from "graphology-layout-force/worker";

const renderer = new Sigma(graph, container, {
  settings: { renderEdgeLabels: true },
});

const layout = new ForceSupervisor(graph);
layout.start();
```

### 2. Export with one call

The `downloadAsImage` function captures the current sigma canvas and triggers a file download:

```typescript
downloadAsImage(renderer, {
  format: "png", // or "jpeg"
  fileName: "graph",
  backgroundColor: "#ffffff",
  cameraState: { x: 0.5, y: 0.5, angle: 0, ratio: 1 }, // optional: reset camera
});
```

### Key concepts

- **`@sigma/export-image`**: satellite package that composites sigma's canvas layers into a single image
- **Format options**: supports PNG (lossless) and JPEG (lossy, smaller files)
- **Camera reset**: optionally capture the full graph regardless of current viewport position
