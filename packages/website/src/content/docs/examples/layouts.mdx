Graph layout algorithms assign positions to nodes. Sigma.js renders whatever positions are stored in the graph, so you can use any layout library that works with [graphology](https://graphology.github.io/).

This example demonstrates three layout strategies:

- **ForceAtlas2** -- a force-directed algorithm that runs continuously in a Web Worker, pulling connected nodes together and pushing unconnected nodes apart.
- **Circular** -- arranges nodes evenly around a circle.
- **Random** -- assigns random positions within the graph's current extent.

Switching between layouts uses `animateNodes()` from sigma's utilities to smoothly interpolate node positions.
