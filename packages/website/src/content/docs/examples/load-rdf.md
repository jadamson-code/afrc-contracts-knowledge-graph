RDF (Resource Description Framework) and JSON-LD are standard formats for representing linked data as graphs of subject-predicate-object triples. Each triple naturally maps to an edge connecting two nodes.

This example will demonstrate how to parse RDF or JSON-LD data, build a graphology graph from the resulting triples, and render it with sigma.js.
