Sigma v4 gives you fine control over label rendering — including font styles, backdrop shapes, positioning, and HTML attachments.

## Label positioning

Labels can be placed relative to their node using the `labelPosition` style property:

```js
styles: {
  nodes: [
    {
      labelPosition: "right", // "right" | "left" | "above" | "below" | "over"
    },
  ],
}
```

You can also set `labelAngle` for rotated labels.

## Backdrops

Backdrops are decorative shapes drawn behind node labels. Configure them via flat `backdrop*` style properties:

```js
styles: {
  nodes: [
    {
      backdropColor: "#ffffff",
      backdropPadding: 6,
      backdropCornerRadius: 4,
      backdropBorderColor: "#cccccc",
      backdropBorderWidth: 1,
      backdropShadowColor: "#00000033",
      backdropShadowBlur: 4,
    },
  ],
}
```

Use `backdropVisibility` to control when backdrops appear (`"visible"` or `"hidden"`), `backdropArea` to choose what the backdrop covers (`"both"`, `"node"`, or `"label"`), and `backdropLabelPadding` to set a separate padding for the label portion.

## Attachments

Label attachments let you display custom HTML content anchored to a node's label. This is useful for tooltips, info cards, or interactive UI:

```js
renderer.on("enterNode", ({ node }) => {
  renderer.setNodeState(node, {
    labelAttachment: {
      position: "below",
      html: `<div class="info-card">${graph.getNodeAttribute(node, "label")}</div>`,
    },
  });
});
```
