The `@sigma/node-image` package adds an image layer that can render photos or SVG pictograms inside nodes.

## Installation

```bash
npm install @sigma/node-image
```

## Rendering images

Import `layerImage` from the package and add it to your node primitives. Each node reads its image URL from the `image` attribute by default:

```typescript
import { layerImage } from "@sigma/node-image";
import Graph from "graphology";
import Sigma from "sigma";

const graph = new Graph();
graph.addNode("a", {
  x: 0,
  y: 0,
  size: 20,
  label: "Alice",
  image: "https://example.com/alice.jpg",
});

new Sigma(graph, container, {
  primitives: {
    nodes: {
      layers: [layerImage()],
    },
  },
});
```

The image is clipped to the node shape (circle by default) and rendered on top of the node fill color.

## Rendering SVG pictograms

To render colored icons instead of photos, use the `"color"` drawing mode. In this mode, image pixels are colorized with the node's color, which works well with single-color SVG icons:

```typescript
import { layerImage } from "@sigma/node-image";

new Sigma(graph, container, {
  primitives: {
    nodes: {
      layers: [
        layerImage({
          drawingMode: "color",
          padding: 0.2,
        }),
      ],
    },
  },
});
```

Set each node's `image` attribute to an SVG URL, and its `color` attribute to the desired tint:

```typescript
graph.addNode("person", {
  x: 0,
  y: 0,
  size: 20,
  color: "#e22653",
  image: "https://icons.getbootstrap.com/assets/icons/person.svg",
});
```

The `padding` option (0 to 1) adds space between the icon and the node edge. A value of `0.2` keeps the icon at 80% of the node diameter.

## Layer options

The image layer accepts these options:

| Option           | Default   | Description                                                  |
| ---------------- | --------- | ------------------------------------------------------------ |
| `drawingMode`    | `"image"` | `"image"` renders as-is, `"color"` colorizes with node color |
| `padding`        | `0`       | Space around the image (0-1 fraction of diameter)            |
| `imageAttribute` | `"image"` | Node attribute to read the image URL from                    |
| `colorAttribute` | `"color"` | Node attribute to read the tint color from (color mode only) |
