The `@sigma/node-piechart` package renders nodes as pie charts, useful for showing categorical distributions on each node.

## Installation

```bash
npm install @sigma/node-piechart
```

## Basic usage

Import `layerPiechart` from the package and define your slices. Each slice has a fixed `color` and a `value` read from a node attribute:

```typescript
import { layerPiechart } from "@sigma/node-piechart";
import Graph from "graphology";
import Sigma from "sigma";

const graph = new Graph();
graph.addNode("a", {
  x: 0, y: 0, size: 20, label: "A",
  positive: 10, neutral: 17, negative: 14,
});

new Sigma(graph, container, {
  primitives: {
    nodes: {
      layers: [
        layerPiechart({
          slices: [
            { color: "#6DDB55", value: { attribute: "positive" } },
            { color: "#7798FA", value: { attribute: "neutral" } },
            { color: "#F05454", value: { attribute: "negative" } },
          ],
        }),
      ],
    },
  },
});
```

Slice proportions are computed automatically from their values. If a node is missing an attribute, that slice gets a value of 0.

## Default color

When all slice values are 0 (or missing), the node renders as a solid circle. Set `defaultColor` to control what color it uses:

```typescript
layers: [
  layerPiechart({
    defaultColor: "#BCB7C4",
    slices: [
      { color: "#F05454", value: { attribute: "negative" } },
      { color: "#7798FA", value: { attribute: "neutral" } },
      { color: "#6DDB55", value: { attribute: "positive" } },
    ],
  }),
]
```

## Fixed-value slices

Slices can also have fixed values instead of attribute-bound ones. This is useful when you want equal-sized segments whose visibility is toggled by whether the attribute is nonzero:

```typescript
slices: [
  { color: "#956b5e", value: { attribute: "slice1" } },
  { color: "#ff44de", value: { attribute: "slice2" } },
  { color: "#71db97", value: { attribute: "slice3" } },
  { color: "#ff813b", value: { attribute: "slice4" } },
]
```

With node data like `{ slice1: 1, slice2: 1, slice3: 0, slice4: 0 }`, only the first two slices would be visible, splitting the pie in half.

## Rotation offset

By default, the first slice starts at the top of the circle. Use `offset` to rotate the starting angle (in radians):

```typescript
layers: [
  layerPiechart({
    offset: Math.PI / 4,
    slices: [/* ... */],
  }),
]
```

The offset can also be read from a node attribute with `offset: { attribute: "piechartOffset" }`.
